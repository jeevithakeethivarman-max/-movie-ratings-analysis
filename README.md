# Movie Ratings Analysis & Genre-based Rating Prediction

## Overview
Analyzed the MovieLens dataset to identify rating trends and predict movie ratings based on genres using Machine Learning.

## Technologies Used
Python, Pandas, Matplotlib, Seaborn, Scikit-learn

## What I Did
- Performed data cleaning and merging of movies and ratings datasets
- Conducted Exploratory Data Analysis (EDA) to find top-rated movies and genre trends
- Built a Linear Regression model to predict movie ratings based on genre
- Achieved MAE of 0.82 and RMSE of 1.03

## Dataset
MovieLens Latest Small Dataset - https://grouplens.org/datasets/movielens/latest/

## Key Insights
- Drama and War genres have the highest average ratings
- Comedy is the most frequently rated genre
- Genre alone explains partial variance in ratings
